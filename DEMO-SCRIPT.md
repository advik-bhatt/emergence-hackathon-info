## If a judge pushes (armor lines)

- **"Is the LLM real?"** — "Yes — OpenAI-compatible client against Nebius Token Factory; set `NEBIUS_API_KEY` and the mode badge flips to LIVE LLM. Without a key it runs the identical 5-step scripted trace over the same tools — that's a reliability feature, and the trace is provably the same either way."
- **"Where does the SQL come from?"** — "Emergence CRAFT's `generate_sql`, recorded live and committed as fixtures — hover any number and you see the natural-language question and the exact SQL. None of it is hand-written."
- **"Why should I trust the narrative?"** — "A hallucination guard parses every number in the narrative and fails the build if any figure wasn't produced by a tool. 131 verified this run."
- **"Isn't the 3D map decoration?"** — "The z-axis is the sort key: height is order-days of broken promise, `gap × orders`. Rio towering next to a calm São Paulo *is* the finding."
- **"Why not just promise longer everywhere?"** — point at the trap section: "That's exactly the optimum we computed — and refused. Padding Rio to 38 makes the #2 market look slower than lanes that are genuinely remote. The queue says fix variance, pad only distance."
- **"What about the gray states?"** — "Excluded by the ≥500-order filter — we don't speak where the data is thin."
- **"Conversion impact?"** — "Unmeasurable — Olist has no clickstream, and the product refuses to claim what it can't measure. That refusal is the product."
